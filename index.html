Firebaseで家族とリアルタイム同期します。誰かの編集が全員のアプリに即反映。

📖 Firebase設定の取得手順 (初回のみ)
  1. Firebase Consoleで「プロジェクトを追加」
  2. プロジェクト作成後、左メニュー「Firestore Database」→「データベースの作成」→ テストモードで開始
  3. 画面上部「⚙ プロジェクト設定」→ 下部「マイアプリ」で Web(</>)アイコン → アプリ名入力 → 登録
  4. 表示される firebaseConfig = { ... } の { }の中身 をコピーして下に貼り付け
